This a Web App for car data analysis and visualization of different comparisons and correlations in the form of block diagrams, histograms, pie charts, etc.

Process Followed:
  1. Data Cleaning
    1. String to number conversion
    2. Null value filtering
    3. Removal of units
  2. Data Analysis
  3. Data Interpretation
    1. Correlation of features
    2. Top car making companies
    3. Cars by car Body type
    4. Relation between Displacement and Price
    5. Cars count by Fuel type
Correlation of features
Description: The graph plots correlation between various features of cars
Logic
  • I have used Seaborn a Python data visualization library built on top of Matplotlib.
  • A heatmap is used which is a graphical representation of data that uses a system of color-coding to represent different values.
Chart
Inference
  • Ex-Showroom price is positively correlated to Displacement.
  • Ex-Showroom Price is Positively Correlated to the number of Cylinders. This means, more the number of cylinders, more the ex-showroom price.
  • The more the number of cylinders in a car, the more will be its displacement. Generally speaking, the higher an engineâ€™s displacement the more power it can create.
  • The number of doors is highly negatively correlated with Displacement.
Top car making companies
Description: It represents brands with the most number of cars in the Indian market.
Logic
  • I've used pie chart to represent brands with the most number of cars in the Indian market.
  • Value_counts() function of pandas is used to count the numbers of cars of specific brand.
Chart
Inference
  • Maruti Suzuki has more car variants than any other company in India.
  • The Top 5 companies with more than car variants in India are Maruti Suzuki, Hyundai, Mahindra, Tata, and Toyota.
  • Sports car variants are very low.
